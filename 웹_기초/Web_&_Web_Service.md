### Web & Web Service

**우리는 웹서비스로 부터 원하는 것을 얻는다.**

#### Request & Response
Request : 요청(클라이언트 : Web) <-> Response : 응답(서버 : Web Service)

* Request :
    * HTML 를~
    1. 갖다 줘 : GET
    2. 처리해 줘 : POST

**예시 :**
나는 네이버를 켜고 = GET(네이버를 갖다줘)
웹툰을 본 다음 = GET(서버야 웹툰 갖다줘)
재미있다고 댓글을 달았다. = 내 댓글 좀 처리해(달아)줘

* (Request & Server의 반대) P2P :
    * Peer - To - Peer : 친구에서 친구로
    * 모두가 서버이자 모두가 클라이언트

#### Web
* www : World Wide Web
    * 인터넷
* 웹 브라우저 : 쌍방향 통신하는 HTML 문서나 파일과 연동하고 출력하는 응용 소프트웨어이다.
    * Chrome, Safari, Edge, FireFox 등등

#### Web Service
* 웹 서비스를 만드는 것은 서버의 역할을 수행하는 프로그램을 만드는 것
